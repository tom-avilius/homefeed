splash

I had this project in my mind ever since I started learning html, css; to be able to create themes for the desktop screen using the web technologies, making it easier for everyone to create and use powerful and productive themes very easily. Unlike rainmeter which uses lua, this project uses html, css and javascript: clearly defining structure, style and behaviour separately.
I have also written several APIs that would make development with splash much easier while at the same I've taken great care in documenting the application and it's APIs.
Splash uses electronJS framework. It was my first time using electronJS and I certainly did encounter various hurdles but in the end I did create something unique and original. The happiness that one receives after being able to create his own product overrides all the hardships.
